# 環境チェック (Environment Check)

このワークフローは、トラブルシューティングの第一歩として、現在のシステムにインストールされている主要なツールのバージョンとパスを確認します。

## 手順

1.  **バージョン確認**
    *   以下のコマンドを順次実行し、出力を表示します。
    *   **Node.js**: `node -v`
    *   **npm**: `npm -v`
    *   **Git**: `git --version`
    *   **PowerShell**: `$PSVersionTable.PSVersion`

2.  **パス確認**
    *   現在の作業ディレクトリを確認します。
    *   `Get-Location`

3.  **レポート**
    *   確認できたツールのバージョン一覧をまとめ、何か問題（古すぎるバージョンや未インストール）がないか診断します。
