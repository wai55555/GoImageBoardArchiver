# 静的サーバー起動 (Serve Static)

このワークフローは、現在のディレクトリにあるHTML/CSS/JSファイルをブラウザですぐに確認するために、簡易的なローカルWebサーバーを立ち上げます。

## 手順

1.  **サーバー起動**
    *   `npx serve` コマンドを使用してサーバーを起動します（インストール不要で高速）。
    *   もし `npx` が使えない場合は、Pythonの `python -m http.server` を代替として使用します。
    *   コマンド:
        ```powershell
        npx -y serve .
        ```

2.  **ブラウザ確認**
    *   サーバーが起動したら、ローカルホストのURL（通常 `http://localhost:3000` または `http://localhost:5000`）をユーザーに伝えます。
    *   可能であれば `browser_subagent` を使ってページが正しく表示されるか確認します。
